생활코딩에서 말한 직접 시련 극복하기란 이런 것일까?
나는 검색을 하여 공부하기 시작했다.

위치 조절법을 배웠다. 내가 직접 찾아 공부하니 더 잘되는 것 같다.
position의 종류는 4가지가 있다. static fixed absolute relative

static은 디폴트 값이다. 이동 할 수 없으며 자기 위치에만 있다.
fixed는 고정이며 스크롤을 해도 그 자리에 고정되어 보여진다.
absolute는 fixed처럼 이동이 가능하지만 따라오진 않는다.
하지만 relative를 부모 element position에 적용하고 자식 element position에 absolute를 적용하면
absolite는 부모 element 안에서 위치를 이동한다. 안에서 밖으로 나갈 수 없다.
